# 미션 - 페어매칭관리 애플리케이션

## 기능 구현 목록

1. 크루별 파일 읽기
2. 크루 정보 저장
3. 기능 선택
   - 예외 사항
     - "1","2","3","Q"가 아니면 예외 발생
       - “잘못된 형식을 입력했습니다.”
4. 페어 매칭 기능
   1. 과정, 레벨, 미션을 선택
      - 예외 사항
        - 올바르지 않은 형식의 입력이면 예외 발생
          - “잘못된 형식을 입력했습니다.”
        - 과정이 “백엔드” 또는 “프론트엔드”가 아니면 예외 발생
          - “잘못된 과정을 입력했습니다.”
        - 레벨이 “레벨1”~”레벨5”가 아니면 예외 발생
          - “잘못된 레벨을 입력했습니다.”
        - 입력한 미션이 해당 레벨에 없는 미션이면 예외 발생
          - “잘못된 미션을 입력했습니다.”
   2. 해당 과정-레벨-미션에 이미 페어 매칭 기록이 존재하는지 판단
   3. 이미 존재한다면
      1. 재매칭 의사 입력
         - 예외 사항
           - “네” 또는 “아니오”가 아니면 예외 발생
             - “잘못된 형식을 입력했습니다.”
      2. “네”인 경우
         1. 매칭 실행
         2. 해당 레벨에서 같은 크루와 매칭 기록이 있다면 재매칭 실행
            - 매칭 시도 3번까지 매칭에 실패한다면 에러 메시지 출력 후 과정, 레벨, 미션 선택으로 이동
              - “매칭에 실패했습니다.”
         3. 매칭 결과 출력
         4. 기능 선택으로 이동
      3. “아니오”인 경우
         - 과정, 레벨, 미션 선택으로 이동
   4. 존재하지 않는다면
      1. 매칭 실행
      2. 해당 레벨에서 같은 크루와 매칭 기록이 있다면 재매칭 실행
         - 매칭 시도 3번까지 매칭에 실패한다면 예외 발생 후 과정, 레벨, 미션 선택으로 이동
            - “매칭에 실패했습니다.”
      3. 매칭 결과 출력
      4. 기능 선택으로 이동
5. 페어 조회 기능
    1. 과정, 레벨, 미션 선택
        - 예외 사항
            - 올바르지 않은 형식의 입력이면 예외 발생
                - “잘못된 형식을 입력했습니다.”
            - 과정이 “백엔드” 또는 “프론트엔드”가 아니면 예외 발생
                - “잘못된 과정을 입력했습니다.”
            - 레벨이 “레벨1”~”레벨5”가 아니면 예외 발생
                - “잘못된 레벨을 입력했습니다.”
            - 입력한 미션이 해당 레벨에 없는 미션이면 예외 발생
                - “잘못된 미션을 입력했습니다.”
            - 해당 과정-레벨-미션에 매칭 기록이 없다면 예외 발생
                - “매칭 이력이 없습니다.”
    2. 매칭 기록 출력
6. 페어 초기화 기능
   1. 모든 페어 기록 초기화
7. 종료
